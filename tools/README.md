# Outils

## Conversion WebP
Ce script convertit les images en WebP et met a jour les sources HTML.

### Installation
```powershell
pip install -r ..\requirements.txt
```

### Utilisation
```powershell
python convert_to_webp.py
```

### Options utiles
```powershell
python convert_to_webp.py --dry-run
python convert_to_webp.py --quality 80
python convert_to_webp.py --recursive
python convert_to_webp.py --force
python convert_to_webp.py --no-html
```
